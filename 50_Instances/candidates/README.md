# 50_Instances/candidates

Pending memory candidates.

Candidates are not durable truth. They require review before promotion.

Use this folder for:

- possible user preferences
- repeated corrections
- low-risk project observations
- candidate decisions

Do not treat candidate records as authoritative context.

Required fields:

- claim
- evidence snippet
- evidence source
- confidence
- last verified date

Candidates must enter `80_Review_Queue/promotion` before promotion.
