# 20_Wiki

Curated durable knowledge.

Wiki notes should be promoted only when they are useful, sourced, and safe to reuse.

Required metadata for curated notes:

- `title`
- `summary`
- `source_status`
- `confidence`
- `last_verified`
- `tags`

Claims without evidence should remain in `80_Review_Queue` or `50_Instances/candidates`.
