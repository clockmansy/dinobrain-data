# DinoBrain Data

This private repository is the DinoBrain data vault.

It stores curated knowledge and operational records for the DinoBrain MCP server. It must not become a dump of raw personal files, raw full conversations, secrets, tokens, or unreviewed imports.

## Folder Map

| Path | Purpose |
| --- | --- |
| `00_Home` | Vault entry points and high-level maps |
| `20_Wiki` | Curated durable knowledge notes |
| `30_Sources` | Source records and verification notes |
| `40_Projects` | Project state, decisions, and handoffs |
| `50_Instances` | Task-derived memory instances |
| `60_Operations` | Operating policies and runbooks |
| `70_Error_Book` | Repeated mistakes, corrections, and prevention notes |
| `80_Review_Queue` | Pending promotion, demotion, and safety reviews |
| `.dino` | App metadata, indexes, and evaluation outputs |

## Safety Rule

When unsure, keep data local and put a summary in `80_Review_Queue`.

Do not commit secrets, raw transcripts, unreviewed private files, or sensitive material.
